Révision PSC1 – Site Web Pédagogique

Ce projet est un site web conçu pour faciliter la révision des notions essentielles du PSC1 (Prévention et Secours Civiques de niveau 1).
Il propose des fiches synthétiques, un quiz interactif, un mode sombre, une interface multilingue et une structure responsive adaptée à tous les écrans.

Ce travail a été développé dans un cadre personnel afin d’améliorer mes compétences en développement web et d’enrichir mon portfolio.


____________________________________________________________________________________________________________________________________________________________
1. Présentation

Le site permet de réviser les principes fondamentaux du PSC1 à travers :

des fiches thématiques interactives,

un quiz dynamique avec explications détaillées,

une interface moderne, adaptable et personnalisable.

L’objectif est de fournir un support clair, simple et efficace aux étudiants et à toute personne souhaitant réviser les gestes de premiers secours.



____________________________________________________________________________________________________________________________________________________________
2. Fonctionnalités principales
Multilingue

Le site est disponible en deux langues :

Français

Anglais
Le choix de la langue est mémorisé automatiquement dans le navigateur.

Mode sombre

Possibilité de basculer entre un mode clair et un mode sombre via un interrupteur visuel.
Le choix du thème est conservé localement.

Fiches pédagogiques

Chaque fiche correspond à une notion essentielle du PSC1 :

Protéger

Alerter

Inconscience

Hémorragies

Malaise

Plaies et brûlures

Le contenu des fiches s’affiche dynamiquement en fonction de la langue choisie.

Quiz interactif

Questions à choix multiples

Validation obligatoire avant de passer à la suite

Explication affichée après chaque réponse

Score final calculé automatiquement

Possibilité de recommencer le quiz

Interface moderne et responsive

Le site est conçu pour fonctionner aussi bien sur ordinateur que sur tablette ou smartphone.
La navigation est simple, intuitive et cohérente.
